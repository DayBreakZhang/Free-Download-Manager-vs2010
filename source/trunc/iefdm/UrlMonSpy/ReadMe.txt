========================================================================
    БИБЛИОТЕКА ДИНАМИЧЕСКОЙ КОМПОНОВКИ. Обзор проекта 
    UrlMonSpy
========================================================================

Эта библиотека DLL UrlMonSpy создана автоматически с помощью 
мастера приложений.

Здесь приведены краткие сведения о содержимом каждого из файлов, использованных 
при создании приложения UrlMonSpy.


UrlMonSpy.vcxproj
    Основной файл проекта VC++, автоматически создаваемый с помощью мастера 
    приложений.
    Он содержит данные о версии языка Visual C++, использованной для создания 
    файла, а также сведения о платформах, настройках и свойствах проекта, 
    выбранных с помощью мастера приложений.

UrlMonSpy.vcxproj.filters
    Это файл фильтров для проектов VC++, созданный с помощью мастера 
    приложений. 
    Он содержит сведения о сопоставлениях между файлами в вашем проекте и 
    фильтрами. Эти сопоставления используются в среде IDE для группировки 
    файлов с одинаковыми расширениями в одном узле (например файлы ".cpp" 
    сопоставляются с фильтром "Исходные файлы").

UrlMonSpy.cpp
    Это основной исходный файл библиотеки DLL.

    При создании этой библиотеки DLL не выполняется экспорт символов. Поэтому
    при ее построении не создается файл .lib. Чтобы определить проект в 
    качестве зависимого от другого проекта, добавьте код для экспорта символов 
    из библиотеки DLL, в результате чего создается библиотека экспорта.
    Также можно открыть диалоговое окно «Страницы свойств», затем в папке 
    «Компоновщик» на странице «Общие свойства» установить для параметра 
    «Пропускать входную библиотеку» значение «Да».

/////////////////////////////////////////////////////////////////////////////
Другие стандартные файлы:

StdAfx.h, StdAfx.cpp
    Эти файлы используются для построения файла предкомпилированного заголовка 
    (PCH) с именем UrlMonSpy.pch и файла предкомпилированных типов 
    с именем StdAfx.obj.
